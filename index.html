Accelerating Image Kernel Operations

Using CUDA, OpenMP, and Hybrid Parallel Approaches

About the Project

This project explores the acceleration of common image processing tasks using three parallel computing techniques: CUDA (GPU-based), OpenMP (multi-threaded CPU), and a Hybrid model that combines both. It aims to demonstrate how parallel programming can significantly reduce computation time when applying image kernels.

Objectives

  • Apply grayscale, blur, and edge detection filters to images
  • Compare the performance of serial vs. parallel implementations
  • Demonstrate the use of CUDA, OpenMP, and hybrid computing in real-world scenarios

Implemented Image Kernels

  • Grayscale Conversion: Converts RGB to intensity values.
  • Gaussian Blur: Smooths images using a weighted average convolution kernel.
  • Sobel Edge Detection: Detects horizontal and vertical edges using gradient-based filters.

Approaches

  • Serial: Basic sequential processing.
  • CUDA: GPU-based parallelism with one thread per pixel, using shared memory for tiles.
  • OpenMP: CPU multithreading to process image regions or multiple images.
  • Hybrid: OpenMP distributes multiple images to threads, each thread invokes CUDA kernels.

Performance Evaluation

The execution time for each method is measured and compared to the serial version to demonstrate the performance gains from using parallel computing techniques.
